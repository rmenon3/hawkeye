## 安装

```ts
// git repo
git clone https://github.com/wangly19/nextui-design-pro.git

// go to projectDir
cd nextui-design-pro/app/next

// install package.json
pnpm install or pnpm i

// run
pnpm run dev

// build
pnpm run build
```
